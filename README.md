# BMI Calculator (JavaScript)

A basic BMI calculator that takes height and weight as input and calculates Body Mass Index.

## Features
- Takes user input for height and weight
- Calculates BMI using standard formula
- Displays result on the page

## Tech Used
- HTML
- CSS
- JavaScript

## Learning Outcome
- Handling form inputs
- Preventing page reload
- Basic math logic in JavaScript
